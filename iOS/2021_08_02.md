# xcode Navigator에서 표시된 알파벳의 의미
Those characters refer to the source control.
|alphabet|meaning|
|--------------|---------|
|?|Unversioned|
|M| Modified|
|A | Added|
|A+ | Moved / renamed|
|U | Newer version of a file on source control|
|I | Item is being ignored|
|! | Item is missing|
